Python/R/Matlab/Octave syntaxes are almost identical, just diverging through some basic language elements:
   
| Python   | R        | Matlab/Octave |
|----------|----------|---------------|
| `a = b`  | `a <- b` | `a = b`       |
| `True`   | `TRUE`   | `true`        |
| `False`  | `FALSE`  | `false`       |
| `None`   | `NULL`   | `[]`          |
| `a.b()`  | `a$b()`  | `a.b()`       |
